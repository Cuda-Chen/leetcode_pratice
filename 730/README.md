# 730. Count Different Palindromic Subsequences
Solution of leetcode problem #730.

# How to Compile and Run
For ordinary version:
```
$ make
$ ./730.out
```

For pthread version:
```
$ make p_thread
$ ./730_pthread.out
```

For CUDA version:
```
$ make cuda
$ ./730_cuda.out
```

For OpenCL version:
```
$ make opencl
$ ./730_cl.out
```
